# Weather App 🌦️

A full-stack weather application built using Node.js, Express.js, HTML, CSS, and JavaScript.

## Features

- Search weather by city
- Real-time temperature
- Humidity details
- Wind speed
- Weather condition
- Weather icons
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- OpenWeatherMap API

## Project Structure

```bash
weather-app

├── public
│   ├── index.html
│   ├── style.css
│   └── script.js
│
├── server.js
├── package.json

```

## Installation

1. Clone repository

```bash
git clone https://github.com/yourusername/weather-app.git
```

2. Install dependencies

```bash
npm install
```

3. Add API key in `.env`

```env
API_KEY=fc89186c07eb1b7a49faa1e92c989767
```

4. Run server

```bash
node server.js
```

5. Open browser

```bash
http://localhost:5000
```
